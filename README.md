# Deloitte Australia - Technology Job Simulation (Forage)

## Overview

This repository contains my solutions for the Deloitte Australia Technology Job Simulation hosted on Forage. The simulation involved completing practical software engineering tasks focused on data transformation and software development planning.

---

## Task 1: Data Transformation

Implemented Python functions to convert two different telemetry data formats into a single unified JSON model.

### Key Features
- Converted telemetry data from multiple JSON formats.
- Parsed location information into a structured format.
- Converted ISO 8601 timestamps into Unix timestamps (milliseconds).
- Passed all provided unit tests.

---

## Task 2: Software Development Proposal

Prepared a professional software development proposal for a real-time manufacturing dashboard.

The proposal includes:
- Project overview
- Functional scope
- Development effort estimation
- Project timeline
- Post-deployment support plan
